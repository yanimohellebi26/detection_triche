# Academic Cheating Detection System

## Tagline
Video and audio analysis system for detecting suspicious behaviour during exams using computer vision and ML.

## Description
A full-stack surveillance assistance system that analyzes exam footage to detect potential cheating behavior. Uses OpenCV for frame-by-frame video analysis and a separate audio anomaly detection pipeline. Features a React frontend for uploading footage and browsing flagged timestamps, with a FastAPI backend orchestrated via Docker Compose.

## Motivation
Assisting human invigilators by automating the tedious task of reviewing hours of exam footage — flagging suspicious moments for human review rather than requiring constant attention.

## Tech Stack
- **Backend**: Python, FastAPI, OpenCV, NumPy
- **Frontend**: React, Vite
- **AI/ML**: Computer vision (OpenCV), Audio anomaly detection
- **DevOps**: Docker Compose

## Key Features
- Video upload with real-time frame-by-frame analysis using OpenCV
- Audio-based anomaly detection pipeline
- Modular backend with separate routers for video and audio analysis
- React frontend for uploading footage and browsing flagged timestamps
- Docker Compose for full-stack orchestration

## Skills Demonstrated
